What is Lua?

Lua is a coding language which originates from Rio de Janeiro in Brazil, created by Roberto Ierusalimschy, Luiz Henrique de Figueiredo, and Waldemar Celes in 1993 while they were part of the Computer Graphics Technology Group at a church. Today, this code is used by thousands of professionals every day on services like Roblox Studio and many more.

Key Features of Lua

  • Lightweight and embeddable
  • Fast execution speed
  • Easy to learn and use
  • Extensible with C/C++
  • Used in game development, scripting, and more

function greet(name) {
  console.log("Hello, " + name + "!");
}
